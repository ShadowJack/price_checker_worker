Описание от производителя
Набор посуды "Travola" состоит из трех кастрюль с крышками вместительностью 10 л, 5,5 л, и 2,5 л, сковороды с крышкой, вока с крышкой, кулинарной лопатки, кулинарной лопатки с прорезями, кулинарной ложки, шумовки, половника и ложки для спагетти.
Посуда выполнена из алюминия с внутренним антипригарным покрытием. Ручки из нержавеющей стали с пластиковыми вставками надежно крепятся к корпусу посуды. Крышки выполнены из термостойкого стекла и позволяют следить за процессом приготовления пищи. Они оснащены отверстием для выпуска пара и металлическим ободом по краю, который позволит предотвратить сколы стекла.
Кухонные приборы выполнены из нейлона, что позволяет использовать их во время приготовления в посуде с антипригарным покрытием.
Посуду можно использовать на всех типах плит, кроме индукционных.

Это идеальный подарок для современных хозяек, которые следят за своим здоровьем и здоровьем своей семьи. Эргономичный дизайн и функциональность набора позволят вам наслаждаться процессом приготовления любимых, полезных для здоровья блюд.
